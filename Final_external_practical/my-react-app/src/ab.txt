1. What is @Autowired?

@Autowired is a Spring annotation used for Dependency Injection (DI).
Spring automatically finds, creates, and injects the required object into your class.

To automatically create and inject the required bean (object).

    @Autowired   // Spring injects StudentService object automatically
    private StudentService studentService;

    @GetMapping("/name")
    public String getName() {

    }
}
